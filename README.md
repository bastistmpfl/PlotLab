# PlotLab

PlotLab ist ein leichtes Web-Tool zum Importieren von SVG/Bildern und zum Erzeugen von G-Code für Plotter.

## Start

1. Projekt klonen oder herunterladen
2. Datei `docs/index.html` im Browser öffnen

## Funktionen

- SVG- und Bild-Import
- Transformieren (Position, Skalierung, Rotation)
- G-Code-Generierung
- Exclusion Zones (Sperrbereiche)
- Presets und Projekt-Speicherung
- 3D-Preview

## Projektstruktur

- `docs/` enthält die gesamte Web-App (HTML, CSS, JavaScript)
